# 便捷操作-VS2022

  • 1 快捷键
    • 1.1 界面操作
    • 1.2 查看
    • 1.3 编辑
    • 1.4 调试
  • 2 代码块

# 1 快捷键

# 1.1 界面操作

打开属性窗口： F4

# 1.2 查看

返回上一光标位置： Ctrl + -

前进到下一光标位置： Ctrl + Shift + -

调用关键字智能提示： Ctrl + J 或 Alt + ->

调用参数提示： Ctrl + Shift + Space

快速切换窗口： Ctrl + Tab

跳转指定行： Ctrl + G

显示/隐藏当前代码块： Ctrl + M + M

折叠所有代码块： Ctrl + M + O

显示所有代码块： Ctrl + M + L

代码自动换行（显示效果）： Ctrl + E + W

查看相关定义： Ctrl + F12 【Esc 退出】

跳转与查询： Ctrl + T

# 1.3 编辑

智能修复（如：修改变量名字同时修改所有引用）： Alt + Enter

重命名： Ctrl + R + R

光标所在行的上面插入一行： Ctrl + Enter

光标所在行的下面插入一行： Ctrl + Shift + Enter

注释：Ctrl + K + C

取消注释：Ctrl + K + U

代码格式化：Ctrl + K + D

# 1.4 调试

启动调试：F5

关闭调试：Ctrl + F5

重启调试：Ctrl + Shift + F5

运行到光标处：Ctrl + F10

单步调试退出当前函数体（从函数体返回）： Shift + F11

# 2 代码块

指令 + 2次 Tab 键

ctor 构造函数
svm 入口函数：static void Main()
cw 控制台打印行：Console.WriteLine()
prop 属性：public int MyProperty { get; set; }
